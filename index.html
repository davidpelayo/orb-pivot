orb-pivot

Interactive pivot grid JavaScript library with drag-and-drop, sorting, filtering, drill-down, and fast React 18 rendering.

Live Demo See Examples View on GitHub
npm install orb-pivot react react-dom

Features

Everything you need to build powerful, interactive pivot tables in the browser.

☰

Drag and Drop

Move fields between row, column, data, and filter axes by dragging. Reconfigure your pivot layout interactively without writing code.

⇅

Sort and Filter

Click any header to sort ascending or descending. Apply visual filters with expression operators: equals, contains, greater than, regex, and more.

🔍

Drill Down

Double-click any aggregated cell to see the underlying detail rows. Explore the raw data behind every number in your pivot table.

Σ

10 Aggregation Functions

Sum, average, count, min, max, product, standard deviation (sample and population), variance (sample and population). Plus custom functions.

∑

Grand and Sub Totals

Show totals for rows and columns independently. Expand and collapse sub total groups for hierarchical data navigation.

🎨

Themes and Export

8 built-in color themes (red, blue, green, orange, flower, gray, black, white), Bootstrap support, and one-click Excel export.

⚡

React 18 Rendering

Powered by React 18 for fast, efficient DOM updates. Works with both UMD (browser) and ES module (bundler) setups.

📊

Multiple Data Fields

Display multiple measures simultaneously. Place data headers on rows or columns to suit your analysis layout.

🛠

Fluent Query API

Query aggregated data programmatically with a fluent, chainable API. Works standalone without rendering any UI.

Code Examples

Get started in minutes. Works with plain HTML, React apps, Node.js scripts, and more.

Browser (UMD) — Full Pivot Table

HTML 
<!-- 1. Include CSS -->
<link rel="stylesheet"
  href="node_modules/orb-pivot/dist/orb.css" />

<!-- 2. Include React 18 -->
<script src="unpkg.com/react@18/umd/react.min.js"></script>
<script src="unpkg.com/react-dom@18/umd/react-dom.min.js"></script>

<!-- 3. Include orb-pivot -->
<script src="node_modules/orb-pivot/dist/orb.min.js"></script>

<div id="pivot"></div>

<script>
  var widget = new orb.pgridwidget({
    dataSource: [
      ['US', 'Laptops', 120, 89999],
      ['US', 'Phones',  200, 159800],
      ['UK', 'Laptops',  90, 67499],
      ['UK', 'Phones',  180, 143820]
    ],
    fields: [
      { name: '0', caption: 'Country' },
      { name: '1', caption: 'Product' },
      { name: '2', caption: 'Qty' },
      { name: '3', caption: 'Revenue',
        dataSettings: { aggregateFunc: 'sum' } }
    ],
    rows: ['Country'],
    columns: ['Product'],
    data: ['Qty', 'Revenue']
  });
  widget.render(document.getElementById('pivot'));
</script>

How it works

  • Load the CSS for pivot table styling
  • Include React 18 UMD builds (required peer dependency)
  • Include the orb-pivot UMD bundle
  • Define your data as a 2D array (rows of values)
  • Map array indices to field captions in fields
  • Assign fields to rows, columns, and data axes
  • Call render() to mount the interactive pivot table

Configuration options

  • width — fixed width in pixels
  • grandTotal — show row/column grand totals
  • subTotal — show/collapse sub totals
  • dataHeadersLocation — 'rows' or 'columns'
  • theme — built-in theme name or Bootstrap

ES Module / Bundler — Programmatic Query

JavaScript 
import orb from 'orb-pivot';
import 'orb-pivot/dist/orb.css';

// Create a pivot grid (data only, no UI)
const pgrid = new orb.pgrid({
  dataSource: salesData,
  fields: [
    { name: '0', caption: 'Country' },
    { name: '1', caption: 'Product' },
    { name: '2', caption: 'Revenue',
      dataSettings: { aggregateFunc: 'sum' } }
  ],
  rows: ['Country'],
  columns: ['Product'],
  data: ['Revenue']
});

// Fluent query API
const q = pgrid.query();

// Grand total
q.val('Revenue');
// => { Revenue: 461119.98 }

// Filter by dimension
q.Country('US').val('Revenue');
// => { Revenue: 249799.99 }

// Multiple fields
q.Country('US').val('Revenue', 'Qty');
// => { Revenue: 249799.99, Qty: 310 }

// Statistical aggregation
q.stdev('Revenue');
q.avg('Revenue');

The Query API

Use orb.pgrid to create a pivot grid without rendering any UI. Then call .query() to get a fluent query object.

Query methods

  • .FieldName(value) — filter by a dimension
  • .val('Field1', 'Field2') — get aggregated values
  • .avg(), .sum() — built-in aggregations
  • .stdev(), .var() — statistics
  • .min(), .max(), .count()

Standalone array query

You can also query arrays directly without creating a full pivot grid:

const result = orb.query(data, fieldDefs)
  .Country('US')
  .Revenue();
// => 249799.99

Custom Formatting and Aggregation

JavaScript 
var config = {
  dataSource: salesData,
  fields: [
    { name: '0', caption: 'Region' },
    { name: '1', caption: 'Product' },
    {
      name: '2',
      caption: 'Revenue',
      dataSettings: {
        aggregateFunc: 'sum',
        // Custom format: currency
        formatFunc: function(val) {
          return '$' + Number(val)
            .toLocaleString('en-US', {
              minimumFractionDigits: 2,
              maximumFractionDigits: 2
            });
        }
      }
    },
    {
      name: '3',
      caption: 'Count',
      dataSettings: {
        // Custom aggregation function
        aggregateFunc: function(
          datafield, intersection, datasource
        ) {
          return intersection.length;
        }
      }
    }
  ],
  rows: ['Region'],
  columns: ['Product'],
  data: ['Revenue', 'Count'],
  // Display options
  grandTotal: {
    rowsvisible: true,
    columnsvisible: true
  },
  subTotal: { visible: true },
  width: 800
};

Custom format functions

The formatFunc receives the aggregated value and returns a display string. Use it for currency, percentages, or any formatting.

Custom aggregation

Replace the built-in aggregation with any function. Your function receives:

  • datafield — the field definition
  • intersection — array of matching row indices
  • datasource — the full data array

Return any numeric value. Common uses: weighted averages, distinct counts, percentile calculations, custom business metrics.

Multiple data fields

List multiple fields in data to show them side by side. Each can have its own aggregation and format function.

Query API in 3 Steps

Query aggregated data programmatically without rendering. Great for backends, data pipelines, and analytics.

1

Create a Pivot Grid

const orb = require('orb-pivot');
const pgrid = new orb.pgrid(config);

Initialize with your data source and field definitions. No DOM required.

2

Build a Query

const q = pgrid.query()
  .Country('US')
  .Product('Laptops');

Chain dimension filters using the fluent API. Each filter narrows the result set.

3

Get Results

q.val('Revenue', 'Qty');
// { Revenue: 89999, Qty: 120 }

q.stdev('Revenue');
// { Revenue: 1377.58 }

Call .val() for default aggregation, or use .avg(), .stdev(), etc.

Contributing

orb-pivot is open source and welcomes contributions. Here's how to get started.

Development Setup

# Clone the repository
git clone https://github.com/davidpelayo/orb-pivot.git
cd orb-pivot

# Use Node 24 LTS (recommended)
nvm use

# Install dependencies
npm install

# Start dev server (Vite, port 3000)
npm run dev

# Run tests (Jest, 146+ tests)
npm test

# Build dist/ (Rollup)
npm run build

Available Scripts

npm run dev         # Vite dev server
npm run build       # Rollup production build
npm test            # Jest test suite
npm run test:watch  # Jest watch mode
npm run test:coverage # Coverage report
npm run lint        # ESLint 9 flat config
npm run lint:fix    # Auto-fix lint issues
npm run format      # Prettier formatting

How to Contribute

  • Fork the repo and create a feature branch
